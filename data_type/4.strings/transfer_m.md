
* \0 ：null（\u0000）
* \b ：后退键（\u0008）
* \f ：换页符（\u000C）
* \n ：换行符（\u000A）
* \r ：回车键（\u000D）
* \t ：制表符（\u0009）
* \v ：垂直制表符（\u000B）
* \' ：单引号（\u0027）
* \" ：双引号（\u0022）
* \\ ：反斜杠（\u005C）

* '\251' // "©"
* '\xA9' // "©"
* '\u00A9' // "©"

*  '\172' === 'z' // true
* '\x7A' === 'z' // true
* '\u007A' === 'z' // true
